# Pilot package

VendorVerdict now has a public pilot package page at `/pricing` and a public pilot request form at `/pilot`.

## Public pages

- `/pricing` explains the paid pilot offer.
- `/pilot` captures a prospect's name, email, company, vendors, use case, and message.
- `/demo` links to the pilot request and sample report.

## Founding pilot

Default public offer:

- Price: from £1,500
- Duration: 4 weeks
- Cases: 10–20 vendor reviews
- Format: guided setup plus review session
- Outputs: saved reports, PDF/Markdown exports, ranked scorecards, due-diligence questions, and pilot summary discussion

## Suggested positioning

Use the paid pilot as the default commercial ask. It is small enough for a first customer to approve, but structured enough to test VendorVerdict on real SaaS buying decisions.
