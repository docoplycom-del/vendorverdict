What these settings affect

Applied automatically

  • New review forms use the default region and data sensitivity.
  • New commercial proposals use the default proposal price and billing wording.
  • Proposal delivery tracking uses the default follow-up date when none is entered.
  • Payment tracking uses the default payment due date and optional payment link.
  • Customer share links and lead follow-up links use the public URL when set.
Secrets stay outside the dashboard.

Passwords, webhook URLs, auth secrets, and API keys should remain in /etc/vendorverdict/vendorverdict.env.

Environment fallback

If a value is reset here, VendorVerdict falls back to the matching environment variable, then to the built-in default.
