Privacy notice

VendorVerdict privacy notice

This page explains how the VendorVerdict pilot instance handles information submitted through the demo, pilot request form, dashboard, and report builder.

Information collected

VendorVerdict may store report requests, generated reports, lead request details, contact information you submit, and operational metadata needed to run the service.

How it is used

The information is used to generate procurement reports, respond to pilot requests, maintain the dashboard, send lead notifications, and operate the production service.

What not to submit

Do not submit passwords, API keys, payment card data, confidential legal documents, private security documents, or unnecessary personal data through public forms.

Storage and access

Production reports and pilot requests are stored in the application database. Dashboard access is protected by authentication. Backups are used for operational recovery.

Contact

For pilot or data questions, use the pilot request form and include only the minimum context needed to understand your SaaS buying workflow.
